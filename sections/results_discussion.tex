To evaluate the performance of SudoSLVRR (MCD), it was compared with the baseline solvers namely Constraint Propagation and Ant Colony System and Constraint Propagation and Multi-Colony ant Optimization with Dynamic Collaborative Mechanism (CD). A total of 45 puzzles, 15 instances each for $9\times9$, $16\times16$, and $25\times25$ Sudoku puzzle, were used and can be downloaded here: https://zenodo.org/records/18812686. Testing was done by executing 100 runs per instance in each puzzle size to accurately measure the success rate, average completion time, and average number of iterations of the solvers.  
To ensure stable convergence and reduce manual tuning, parameters ($\rho$, $\rho_{BVE}$, $\xi$, $E^*(P)$,$con^*$) were drawn from prior ACO and DCM-ACO studies. The experiment was conducted using 3 threads on an AMD Ryzen 5 7530U (16GB RAM) with a 100-iteration interval for early communication ($iter < 200$) and a 10-iteration interval after 200 iterations. 

From the experimental results, we can observe that SudoSLVRR outperforms CA and CD solvers. Although single-threaded CD solver internally balances exploration and exploitation, it still lacks cross-region exploration. By implementing the dual-topology communication, SudoSLVRR was able to cover more regions of the solution space as ring-based exchange propagates locally promising solutions and randomized best-so-far sharing allows cross-region exploration.Thus, multi-level cooperation becomes necessary when solving combinatorial search spaces like $25\times25$ Sudoku.
